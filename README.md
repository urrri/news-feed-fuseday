News feed fuseday
=====================

A news reader application that combines [MobX](https://mobxjs.github.io/mobx) with [React](https://facebook.github.io/react).
Supports ES6 and JSX compilation through babel.

### Run the example

```
npm install
npm start
open http://localhost:3000
```
